# Lunar Diary
## A beautiful space-themed mood journal with moon phases!

## Features
* Stats Overview
* 8 Moon Phases to Choose
* Heidi Quotes
* Add/View/Delete thought
* Data saving(localStorage)

## Technologies used
* React
* CSS
* Lucida Icons
